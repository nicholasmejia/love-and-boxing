# Tofu Riddles + KD-Gated Sub-Decks Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Refactor `DialogueDeck` into three knockdown-gated sub-decks driven by `_knockdowns.count()`, then author Tofu's first nine picture-riddle prompts (text-bodied for now; image-swap is a future per-`.tres` edit).

**Architecture:** A new `DialogueDeckResource` resource type holds three exported `Array[DialoguePrompt]` fields, one per knockdown level. `DialogueDeck` is refactored to track parallel `_remaining` arrays per tier with a single active-tier index. `gameplay.gd` loads the deck via the existing `DifficultyConfig.dialogue_deck_path` field (with a fallback to Tofu's deck when empty/missing) and pushes the new active tier into the deck after every knockdown increment and round-end decrement.

**Tech Stack:** Godot 4.4 (GDScript, Resource subclasses, `.tres` text format), GUT 9.x for unit tests.

**Reference spec:** `docs/superpowers/specs/2026-05-18-tofu-riddles-design.md`

Replace the entire contents of `scripts/game/dialogue_deck.gd` with:

```gdscript
class_name DialogueDeck
extends RefCounted

const TIER_COUNT := 3

var _source: Array = [[], [], []]
var _remaining: Array = [[], [], []]
var _active_tier: int = 0
var _rng := RandomNumberGenerator.new()

func _init() -> void:
	_rng.randomize()

func seed_rng(seed_value: int) -> void:
	_rng.seed = seed_value

func load_tier(tier: int, prompts: Array[DialoguePrompt]) -> void:
	var idx := _clamp_tier(tier)
	_source[idx] = prompts.duplicate()
	_remaining[idx] = prompts.duplicate()

func set_active_tier(tier: int) -> void:
	_active_tier = _clamp_tier(tier)

func draw() -> DialoguePrompt:
	var pool: Array = _remaining[_active_tier]
	if pool.is_empty():
		_refill(_active_tier)
		pool = _remaining[_active_tier]
	if pool.is_empty():
		return null
	var idx := _rng.randi() % pool.size()
	return pool.pop_at(idx)

func reset() -> void:
	for tier in range(TIER_COUNT):
		_refill(tier)

func _refill(tier: int) -> void:
	_remaining[tier] = (_source[tier] as Array).duplicate()

func _clamp_tier(tier: int) -> int:
	return clamp(tier, 0, TIER_COUNT - 1)
```

Note: `_source` and `_remaining` are declared `Array` (not `Array[Array]`) because GDScript 4 does not allow nested typed-array literals as defaults. The runtime contents are still typed arrays once `load_tier` populates them; the typed signature on `load_tier(prompts: Array[DialoguePrompt])` is the type guarantee for callers.
